# Sibling-Snowdown
A chill top-down snowball fight between two siblings made for the Cozy Winter Jam 2023, a 3-day game jam. 
Play with a friend (or enemy) in local multiplayer on the same keyboard. May the best sibling win! 

## Controls: 

### Player 1

Move - WASD 

Throw Snowball - C 

Reload - V 

### Player 2

Move - Arrow Keys

Throw Snowball - K 

Reload - L
